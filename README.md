# pokemon

## Getting Started

### Prerequisites
Ensure you have the following installed:
- Node.js
- npm or yarn

### Installation
1. Clone the repository:
   ```sh
   https://github.com/arunmauryaosm/pokemon.git
2. Navigate into the project directory:
   ```sh
   cd pokemon
3. Install dependencies
   ```sh
   npm install
4. Run the project
   ```sh
   npm run dev
